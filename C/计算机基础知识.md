# 概述
## C语言优点
- 效率高
- 控制能力强
- 程序可移植
- 通用性强,使用广泛
## 编译器
是把代码转换成计算机内部指令的程序
## 编程过程
编译器将源代码编译为目标代码,通过链接器将库代码和启动代码链接起来,从而成为机器可以直接读取的二进制代码(指令集)

## 程序结构
一般函数体和函数头组成
**简而言之，一个简单的C程序的格式如下：**
```C
#include <stdio.h>
int main()
{
     语句
     return 0;
}
```
（大部分语句都以分号结尾。）
## 与python不同
- 固定的输入输出设置
- 声明变量
- 函数体内需要打分号;

## 自定义函数
声明(确定返回值,参数) → 定义(编写函数定义) → 调用(执行函数功能)
`void a(void)`中前面的`void`表示没有返回值，后面`void`表示没有参数,可以省略不写

字符类型和占内存的字节大小有关
变量分为**全局变量**和**局部变量**
